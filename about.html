Plyr

This is the plugin that we didn't cover in class. It allows you to play videos from Youtube and Vimeo as well as regular videos and audios. To use it you need to use either a div, video or audio tag. Then you need to assign an id and a datatype. For both Youtube and Vimeo you create a div that needs to have a data-video-id tag with the appropriate video id (this can be found in the url of a video). Audio and video both need a source and a type (for example audio/mp3). Of course you need to link the appropriate javascript and css files. Lastly you need a single line of javascript to set up the video/audio using the id you assigned earlier.
